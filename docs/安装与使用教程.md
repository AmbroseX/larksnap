# larksnap 安装与使用教程（普通用户版）

> 面向不写代码的普通用户。跟着做，五分钟就能装好用起来。

## 这是什么

larksnap 是一个装在 Chrome / Edge 浏览器里的小插件，用来：

- **把飞书 / Lark 文档一键存到本地**：导出成 Markdown、PDF、HTML，图片和附件一起打包带走。
- **把任意网页转成 Markdown**：整页或选中一段都行，还能解除"不让复制/右键"的网页限制。

全程在你自己的浏览器里完成，**不上传任何内容、不用注册、不用申请权限**。

---

## 一、安装（不用写代码）

### 第 1 步：下载安装包

打开发布页，下载最新版的压缩包：

👉 https://github.com/AmbroseX/larksnap/releases

在最新一条 Release 下面的 **Assets（附件）** 里，点 `larksnap-vX.X.X.zip` 下载（`X.X.X` 是版本号，选最新的那个）。

> 页面里可能还有一个 `.crx` 文件——那是给公司统一下发用的，新版 Chrome 出于安全**默认禁止普通用户直接拖进去安装**。所以请用上面的 `.zip`，别用 `.crx`。

### 第 2 步：解压到一个固定文件夹

把下载好的 zip **解压**出来，放到一个你不会随手删掉的位置（比如"文档"目录下新建一个 `larksnap` 文件夹）。

> ⚠️ 重要：这个文件夹装好后**不能删、不能移动、不能改名**，否则插件会失效。别放在"下载"临时区或桌面临时清理的地方。

### 第 3 步：加载到浏览器

以 Chrome 为例（Edge、360、Brave 等 Chromium 内核浏览器步骤一样，只是地址不同）：

1. 在浏览器地址栏输入 `chrome://extensions` 回车（Edge 用 `edge://extensions`）。
2. 打开右上角的 **「开发者模式」** 开关。
3. 点左上角 **「加载已解压的扩展程序」**。
4. 选中第 2 步解压出来的文件夹（**要选到里面有 `manifest.json` 文件的那一层**）。
5. 列表里出现 "larksnap" 就装好了。
6. 点浏览器工具栏的 **拼图图标 🧩 → 把 larksnap 后面的图钉点亮**，让它常驻在工具栏上，方便点。

> 每次开浏览器可能会弹一个"要停用开发者模式扩展吗"的提示，点 **保留 / 取消** 即可，不影响使用。

---

## 二、用之前：先登录飞书

larksnap 是**借用你浏览器里已经登录的飞书**来干活的。所以第一次用之前，先在浏览器里像平常一样**登录一次飞书 / Lark**（公司自建域名的飞书也可以）。登录过一次后就不用每次都登。

---

## 三、用法一：导出飞书文档

1. 打开任意一篇飞书文档页面（普通文档、知识库 wiki、表格都行）。
2. 点工具栏上的 **larksnap 图标**，浏览器右侧会弹出一个侧边栏。
3. 侧边栏顶部会显示**识别到的文档标题**——看到标题就说明认出来了。
   - 如果是**公司自建域名**的飞书，第一次会提示"授权该网站"，点一下授权按钮即可。
4. 在下面的按钮里点你要的动作：

   | 想要什么 | 点这个 |
   |---|---|
   | 存成 Markdown（图片自动打包成 zip） | 导出为 Markdown |
   | 存成 PDF | 导出为 PDF |
   | 存成单个网页文件 | 导出为 HTML |
   | 只下载文档里的图片和附件 | 导出附件 |
   | 存一份离线快照，以后没网也能看 | 缓存到本地 |

5. 文件会自动下载到浏览器的**默认下载目录**。

> 如果你公司关闭了飞书官方导出功能，插件会先弹一个提示告诉你"这是在绕过该限制"。**请在获得授权的前提下使用。**

---

## 四、用法二：把任意网页转成 Markdown

在**非飞书的普通网页**上，larksnap 有两个入口，随便用哪个：

- **点插件图标**：侧边栏会自动切换成"网页复制"视图。
- **在页面上点右键**：菜单里直接有相关选项。

能做这些事：

| 想要什么 | 怎么做 |
|---|---|
| 整个网页转 Markdown | 点"整页转 Markdown"，可以**复制**到剪贴板，也可以**下载成 `.md` 文件** |
| 只要选中的一段 | 先用鼠标选中文字 → 右键"选中内容转 Markdown" |
| 遇到不让复制 / 不让右键的网页 | 打开"解除复制限制"，页面立刻能选能复制；关掉就恢复原样 |
| 选中就自动复制 | 在设置里开启"选中文字自动复制"，之后选中一段字就自动进剪贴板 |
| 复制网页链接 | "复制本页链接"或"复制全部标签页"，可选纯链接或带标题的格式 |

> 网页复制不需要提前授权任何网站——用右键菜单时，浏览器会在你点击的那一下临时授予当前页权限。

## 五、设置在哪里改

两种方式打开设置页：

- 在插件图标上点**右键 → 选项**；或
- 到 `chrome://extensions`，找到 larksnap → **详情 → 扩展程序选项**。

里面可以调：图片的处理方式、网页"选中自动复制"的最小字数和格式（纯文本 / Markdown）、"复制全部标签页"的输出格式等。

---

## 六、怎么升级到新版本

1. 到发布页下载新版的 `zip`，解压后**覆盖**原来的文件夹（或解压到新文件夹）。
2. 打开 `chrome://extensions`，点 larksnap 卡片上的**刷新按钮 🔄**。
3. 如果换了新文件夹，就重新"加载已解压的扩展程序"选新文件夹。

---

## 七、常见问题

- **侧边栏没识别出文档 / 一片空白？** 刷新一下文档页面再打开侧边栏；确认你打开的是文档正文页，而不是文件夹目录页。
- **提示未登录？** 在浏览器里正常登录一次飞书，再回来重试。
- **公司自建域名的飞书不工作？** 侧边栏里会有一个"授权该网站"的按钮，点一下授予权限即可。
- **工具栏上找不到图标？** 点浏览器右上角的拼图图标 🧩，把 larksnap 的图钉点亮固定。
- **加载时报错"清单文件缺失或无效"？** 说明选错了文件夹——要选到**里面直接有 `manifest.json`** 的那一层。
- **插件突然失效了？** 多半是安装文件夹被删、被移动或改名了。把文件夹放回原处，或重新下载解压再加载一次。
- **更新后没生效？** 到 `chrome://extensions` 点刷新，或先"移除"再重新加载。

---

## 八、关于隐私

larksnap 全程在你自己的浏览器里运行：文档内容、登录状态、Cookie **都不会发给任何服务器**，也没有后台、不用注册、不用申请 API。详细说明见项目里的 [`PRIVACY.md`](../PRIVACY.md)。

> 本工具仅供合法、获授权的文档导出与个人备份。请确保你对导出的文档有相应权限，并遵守所在组织的数据管理规定与飞书 / Lark 的服务条款。
